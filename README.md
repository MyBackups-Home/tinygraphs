Stack
======

* Go

Installation

*   `go get github.com/taironas/greentros`
*   `cd $GOPATH/src/github.com/greentros`
*   `go get`
*   `export PORT=8080`

Run App

    > pwd
    $GOPATH/src/github.com/taironas/greentros
    > greentros
    2014/11/19 22:23:57 Listening on 8080
